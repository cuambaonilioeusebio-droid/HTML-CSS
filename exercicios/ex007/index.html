Codigo-fonte em uma linha

O comando System.out.println("Olá mundo") é escrito em linguagem java.

Codigo-fonte comprido

				
	public class ExemploJava{
		public static void main(String args){
		
			mensagem("Esse é o meu texto inserido");
		
		}
		public static void mensagem(String texto){
		
			System.out.println("Texto inserido:\n"+texto);
		
		}
	}
